Day 1: Introduction to Robotics and Electronics

  • Basics of Voltage, Current, Resistance; DC vs. AC
  • Activity: Build an LED + resistor circuit
  • Outcome: Understand simple circuits and key concepts

Day 2: Components and Sensors

  • Resistors, LEDs, Relays, IR and Ultrasonic Sensors
  • Activity: Control lamp using relay; IR sensor detection
  • Outcome: Interface sensors and read input

Day 3: Getting Started with ESP32

  • ESP32 pinout, boot/reset, IDE setup
  • Activity: Blink LED; host web server
  • Outcome: Configure ESP32 and connect via browser

Day 4: Motor Control with PWM

  • H-Bridge drivers, PWM basics
  • Activity: DC motor direction control; Servo sweeping
  • Outcome: Motor/servo control using PWM

Day 5: Wireless Communication with ESP32

  • Bluetooth Low Energy (BLE), REST APIs
  • Activity: Mobile app to toggle LED; HTTP/JSON control
  • Outcome: BLE/Wi-Fi based control

Day 6: Capstone Project – Bluetooth Car

  • Assemble car; Upload debug code
  • Focus: Chassis, steering, BLE firmware
  • Outcome: Working robotic BLE car

Day 7: Capstone Project – Smart Home Automation

  • Build 2-channel smart switch with voice control
  • Cloud connectivity with Google Home/Alexa
  • Outcome: Voice-controlled home automation
