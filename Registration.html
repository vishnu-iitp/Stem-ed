7-Day Workshop Breakdown

Day 1: Introduction to Robotics and Electronics
  • Topics: Basics of Voltage, Current, Resistance; DC vs. AC
  • Activities: Build a basic LED + resistor circuit on a breadboard
  • Outcomes: Understand simple circuits and key electronics concepts
Day 2: Components and Sensors
  • Topics: Resistors, LEDs, Relays, IR and Ultrasonic Sensors
  • Activities: Control a lamp using a relay; IR sensor obstacle detection
  • Outcomes: Interface electronic components and read sensor input
Day 3: Getting Started with ESP32
  • Topics: ESP32 pinout, boot/reset, IDE setup
  • Activities: Blink LED sketch; Host a simple web server
  • Outcomes: Configure ESP32 and interact through a browser
Day 4: Motor Control with PWM
  • Topics: H-Bridge motor drivers, PWM basics
  • Activities: DC motor speed/direction control; Servo sweeping
  • Outcomes: Use PWM for motor control and servo positioning
Day 5: Wireless Communication with ESP32
  • Topics: Bluetooth Low Energy (BLE), REST APIs
  • Activities: Use a mobile app to toggle an LED; Send HTTP JSON commands
  • Outcomes: Enable BLE and Wi-Fi based device control
Day 6: Capstone Project 1 – Bluetooth Car
  • Activities: Assemble hardware; Upload and debug control code
  • Focus: Chassis design, steering mechanics, BLE firmware
  • Outcomes: Fully functional BLE-controlled robotic car
Day 7: Capstone Project 2 – Smart Home Automation
  • Activities: Build a two-channel smart switch; Enable voice control
  • Focus: Cloud connectivity with Google Home/Alexa
  • Outcomes: Working home automation system with voice commands

Who Can Join?

This workshop is designed for students from Class 7 to Class 12. The content is tailored to be understandable and engaging for all levels, with separate attention and support for different age groups when necessary.

Resources (If Any Needed)

We will bring most of the required materials. However, we request the school to provide:

  • A classroom
  • Projector or smart board

All other hardware (Arduino kits, sensors, laptops, etc.) will be provided by our team during the session.
